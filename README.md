# 抖音商品评价 AI 分析看板

一个纯前端实现的电商评价数据分析工具，支持批量导入 Excel/CSV，调用大模型 API 进行 AI 自动打标，并提供多维度的数据可视化看板。无需后端服务器，打开网页即可使用。

## 核心功能

### 一、基础操作页

| 功能 | 说明 |
|------|------|
| **批量导入** | 支持同时上传多个 `.xlsx`、`.xls`、`.csv` 文件，自动合并解析 |
| **数据预处理** | 对导入的数据进行标准化、清洗和格式化，确保后续分析准确性 |
| **AI 自动打标** | 调用配置的大模型 API，对每条评价自动进行「评价分类」和「评价等级（好评/中评/差评）」标注 |
| **系统配置** | 支持自定义店铺列表、大模型 API 接口、AI 提示词模板 |

### 二、数据看板页

提供 **6 大核心 KPI** 与 **5 大分析模块**，所有图表均支持实时联动筛选。

#### 核心 KPI 指标
- 总评价数
- 好评率 / 差评率
- 综合评分
- 疑似水军数量
- 有内容率

#### 多维度筛选器
可按 **门店、评价分类、评价等级、商品名称、商品规格、省份、发货仓库、日期范围** 自由组合筛选。

#### 分析模块

| 模块 | 包含图表 |
|------|---------|
| **总体概览** | 评价等级分布、评分星级分布、各店铺好评率 vs 差评率对比、好评/差评省份地图分布、好评/差评发货仓库占比 |
| **趋势分析** | 每日评价等级趋势（堆叠折线图）、每日差评趋势 |
| **关键词分析** | 正面关键词频率 TopN、负面关键词频率 TopN、商品名称好评/差评分布、商品规格好评/差评分布 |
| **疑似水军刷评** | 自动识别同一用户发布 ≥5 条好评或差评的疑似水军行为，分别展示好评水军与差评水军 |
| **评价明细** | 看板全量数据表格展示，支持导出 Excel |

### 三、数据持久化

- **导出数据库**：将当前所有数据导出为 `.db` 文件，下次可直接导入恢复
- **导入数据库**：支持导入之前导出的 `.db`、`.sqlite`、`.sqlite3` 文件
- **导出 Excel**：看板评价明细支持一键导出为 Excel 表格

## 使用指南

### 1. 首次使用
1. 打开网页，进入「基础操作页」
2. 在右侧「系统与大模型配置」中填写你的大模型 API 地址和 Key
3. 点击「上传多表格」导入电商平台的评价数据文件
4. 点击「预处理」完成数据清洗
5. 点击「AI 自动打标」等待批量分析完成

### 2. 查看分析结果
1. 切换至「数据看板页」
2. 使用顶部筛选器组合筛选感兴趣的数据维度
3. 点击不同 Tab 查看总体概览、趋势分析、关键词分析、疑似水军和评价明细

### 3. 保存工作进度
- 点击左侧侧边栏的「导出数据库」图标，将当前数据保存为 `.db` 文件
- 下次使用时点击「导入数据库」图标即可恢复全部数据

## 本地运行

本项目为纯静态网页，无需构建工具，直接在浏览器打开 `index.html` 即可使用。

```bash
# 方式一：直接双击打开 index.html

# 方式二：使用本地静态服务器（推荐，避免跨域限制）
# Python 3
python -m http.server 8080

# Node.js
npx serve .
```

---

## 注意事项

- **AI 打标功能**需要配置有效的大模型 API 才能正常使用
- 数据全部存储在浏览器本地（IndexedDB + sql.js），清理浏览器缓存会导致数据丢失，请及时「导出数据库」备份
- 建议单次导入数据量控制在合理范围内，以保证浏览器端处理性能
